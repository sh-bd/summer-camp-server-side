## Run Locally

Clone the project

```bash
  git clone https://github.com/programming-hero-web-course1/b7a12-summer-camp-server_side-sh-bd.git
```

Go to the project directory

```bash
  cd b7a12-summer-camp-server_side-sh-bd
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  nodemon index.js
```
